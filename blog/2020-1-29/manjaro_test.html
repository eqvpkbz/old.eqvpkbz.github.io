manjaro系统初探

其实manjaro各方面都挺香的

我觉得可以吊打ubuntu

好的，接下来记录一下安装的过程

首先和ubuntu的刻录软件一样，用的是rufus

刻录好之后安装就行

更换国内源

1
2
3
sudo pacman -Syy
sudo pacman-mirrors -i -c China -m rank  #选清华源即可
sudo pacman -Syyu

添加Arch的国内源

1
2
sudo pacman -S vim
sudo vim /etc/pacman.conf

在文件末尾添加

1
2
3
[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

然后

1
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

安装git

1
2
3
sudo pacman -S git
git config --global user.name "Eqvpkbz"
git config --global user.email "2581505024@qq.com"

安装zsh

1
2
sudo pacman -S zsh
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

输入法

1
2
3
sudo pacman -S yay fcitx-sogoupinyin fcitx-im fcitx-configtool
sudo pacman -S fcitx-skin-material yay
yay -S fcitx-qt4

设置中文输入法环境变量，编辑~/.xprofile文件，增加下面几行(如果文件不存在，则新建)

1
2
3
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

打开fcitx配置-附加组件, 取消选定经典皮肤

Skype

1
2
sudo pacman -S gnome-keyring
yay -S skype

然后登录skype的时候，输完密码会提示创建密钥环，此时不输任何密码，直接继续(不加密操作，可以让后面登录不用输密码)

值得一提的是KDE Wallet不能取消，不然每次网络密码要重输

TIM

安装TIM

1
2
3
sudo pacman -S gnome-settings-daemon
sudo  cp  /etc/xdg/autostart/org.gnome.SettingsDaemon.XSettings.desktop   ~/.config/autostart
yay -S deepin-wine-tim

然后安装即可

修改内置IE浏览器

因为deepin-wine内部似乎还打包了一个ie浏览器，但是什么内容都显示不出来

所以我们可以把ie浏览器改成系统的浏览器，比如google-chrome

  1. 找到iexplore.exe的位置，一般在/home/用户名/.deepinwine/Deepin-TIM/drive_c/Program Files/Internet Explorer/，在此处打开终端

先备份

1
mv iexplore.exe iexplore.exe.backup
  1. 找到自己正在用的浏览器的位置，创建软连接 
1
ln -s $(which google-chrome-stable) iexplore.exe
